    iROSLink

    Turn your iPhone Pro into a ROS2 sensor bridge. Stream LiDAR, camera, and IMU data over WiFi — then let the phone navigate a robot autonomously.

    Try it
    Cardboard prototype build
    Autonomous vacuum cleaning
    Live sensor data in RViz2
    📡

    Sensor Bridge

    ROS2 jazzy compatible — IMU, GPS, camera, LiDAR depth, and point cloud — published straight from your iPhone over Zenoh.

    🗺️

    Autonomous SLAM

    On-device RTAB-Map builds 2D occupancy grids run entirely on the phone.

    📊

    Foxglove Ready

    Built-in WebSocket server streams all topics directly to Foxglove Studio. No bridge, no Docker, no ROS2 on your laptop needed.
